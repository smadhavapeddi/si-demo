Glossary

Plain-language explanations for the terms you'll see on screen.

The big picture
Inference
When an AI model thinks and gives you an answer. Every prompt you send is one inference.
Serverless inference
You just use the AI. DigitalOcean handles the GPUs, scaling, and plumbing. You pay only for what you use.
Router router:*
A smart switchboard. It reads your prompt, figures out the task, and sends it to the best model for the job.
Model
The actual AI brain (Claude, Llama, GPT, Gemma…). Some are big and smart, some are small and fast, some specialize in code or writing.
What you put in
Prompt
What you ask the AI — the text you type in the box.
Token
How AI counts text. Roughly ¾ of a word. "Hello world" is about 2 tokens. You're charged per token.
Max tokens
A limit on how long the answer can be — like saying "keep it under 200 words." Protects you from runaway replies and surprise bills.
Temperature
How predictable or imaginative the answer is. Low (0 – 0.3) = serious and consistent — same answer every time. Good for classifying, extracting data, or anything factual. High (0.7 – 1) = playful and varied. Good for brainstorming, writing, or creative ideas.
Context window
How much the AI can read in one shot — its short-term memory for this request. Bigger window = can digest longer documents.
What you get back
Tokens in / out
in = how much you asked. out = how much the AI wrote back. Both count toward your bill.
Latency
How long it takes to get the answer back. Small models are fast; big models on long prompts are slower.
Finish reason
stop = the AI finished naturally. length = it ran out of room — bump up Max tokens if the answer got cut off.
Reasoning content
Some models show their work — like a student writing out steps before the final answer. If the answer is blank, we show this so you can see what the model was thinking.
Beyond text
Image generation
Same platform, same key — but the model draws instead of types. Response is a b64_json image you can show or save.
Multimodal
Text, images, embeddings, and audio all on one inference platform. One bill, one SDK, one account.
Under the hood
Chat completions
The standard API shape. If you've used OpenAI's API, DigitalOcean's /v1/chat/completions looks identical — same SDKs, no code rewrites.
Image generations
The image endpoint: /v1/images/generations. Same auth, same account — just a different URL for visuals.
